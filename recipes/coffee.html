Java Frappuccino

Description

If you followed my previous recipes, you have made some delicious raspberry pie and amazing cookies. Now it is time to brew some fresh Java coffee

Ingredients

  • 1 Notepad
  • ½ cup brewed Source Code or 2 plain texts
  • 1 class A
  • 1 cup ice cubes
  • ¼ cup main method - public static void main(String args[]){}
  • 1 System.out.println(“Hello World”)
  • A working folder
  • 1 tablespoon "Hello World" cream and sprinkles

Steps

  1. Brew your Source Code and let it cool. Put it in the freezer for 10 minutes to cool it faster.
  2. Pour the Source Code, class A, ice cubes, main method, and “Hello World” into a high-speed Notebook. Press the pulse button a few times to get the blending started, then blend on high speed for 20 to 30 seconds.
  3. Try your Java Frappuccino and see what you think! Depending on your Notebook, you may need to stir and continue blending. Blend until the ice there aren’t any ice chunks left.
  4. Pour a little Command Prompt syrup down the sides of your tall glass. Then pour the Java Frappuccino into the glass.
  5. Top with "Hello World" cream and sprinkles for an extra decadent treat!
